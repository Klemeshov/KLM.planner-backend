# KLM Planner backend

KLM Planner is a web application designed to help users plan and manage their tasks and time efficiently.

Backend using NestJS framework.

## node

recommended `v22.11.0` or higher

## Project setup

```bash

npm install
```

## Compile and run the project

#### development

```bash

npm run start
```

#### watch mode

```bash

npm run start:dev
```

#### production mode

```bash

npm run start:prod
```

## Run tests

#### unit tests

```bash

npm run test:unit
```

#### e2e tests

```bash

npm run test:e2e
```

#### test coverage

```bash

npm run test:cov
```
